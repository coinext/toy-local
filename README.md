# toy-local

## oauth2 실행
```
    > cd toylocal-oauth
    > gradle bootRun
```

## server 실행
```
    > cd toylocal-server
    > gradle bootRun
```

## front 실행
```
    > cd toylocal-front
    > yarn build
    > yarn start
    // or
    > yarn dev
```
